SEO Page Templates
Project: משתחררים (Mishtachrerim)

This document defines the SEO page templates used by the product.

The purpose of this document is to ensure that every SEO page:

• follows a consistent structure
• targets a specific search intent
• connects to the decision engine
• drives users toward the quiz and career paths

The public site content is written primarily in Hebrew.

---

CORE PRINCIPLE

Every SEO page must do two things:

1. Provide real value
2. Move the user one step closer to a decision

No page should exist only as content.

Each page must connect to at least one of:

Checklist
Quiz
Career Path
Courses

---

SEO PAGE TYPES

The system supports five SEO page types:

1. Guide Articles
2. Intent Pages
3. Career Path Pages
4. Course Detail Pages
5. Editorial Course Comparison Pages

INTENT PAGE TEMPLATE

Intent pages answer specific questions users search in Google.

Examples:

איך להיות QA Tester
כמה מרוויח Data Analyst
איך להיכנס להייטק בלי תואר

URL examples:

/career/how-to-become-qa
/career/qa-salary
/guides/how-to-enter-tech-without-degree

Page structure:

H1
Specific user question

Example:

איך להיות QA Tester

Introduction

Provide a short direct answer.

H2
מה עושה QA Tester

Explain the job.

H2
למי המקצוע מתאים

Describe personality fit.

H2
כמה מרוויח QA Tester

Provide salary range.

H2
איך מתחילים

Possible paths:

לימוד עצמי
קורס
עבודה התחלתית

H2
קורסים שיכולים לעזור

Link to course pages.

CTA

לא בטוחים אם QA מתאים לכם?

התחילו את השאלון.
