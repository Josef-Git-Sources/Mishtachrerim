SEO Site Map

Project: משתחררים (Mishtachrerim)



This document defines the SEO structure of the site.



The site is designed as a \*\*decision engine with SEO entry points\*\*.



SEO pages bring users from search engines into the product flow.



Core funnel:



Google Search

→ SEO Page

→ Quiz

→ Career Recommendation

→ Courses

→ Conversion



The following sitemap defines the initial SEO surface of the product.
