# INTENT_PAGE_STRATEGY.md

Intent Page Strategy
Project: משתחררים (Mishtachrerim)

This document defines the strategy for creating and scaling intent-based SEO pages for the Mishtachrerim platform.

Intent pages target specific search queries related to professions and career paths.

They allow the platform to capture a large number of long-tail search queries while guiding users toward career exploration and training decisions.

---

# 1. What Are Intent Pages

Intent pages are SEO pages designed to match specific search queries related to a profession.

Examples of queries:

how to become a QA tester
qa tester salary
is qa a good career
how to become data analyst
digital marketing salary

Each of these queries reflects a specific user intent.

Intent pages answer the query and guide the user toward the career decision funnel.

---

# 2. Relationship to Career Pages

Career pages are the central hub for each profession.

Example:

/career/qa-tester

Intent pages support the career page by targeting additional search queries.

Example intent pages:

/career/how-to-become-qa
/career/qa-salary
/career/is-qa-good-career

All intent pages should link back to the main career page.

Structure:

Intent Page

↓

Career Page

↓

Courses

---

# 3. Intent Page URL Structure

Intent pages follow the pattern:

/career/[intent-slug]

Examples:

/career/how-to-become-qa
/career/qa-salary
/career/how-to-become-data-analyst
/career/data-analyst-salary

The slug should reflect the search query as closely as possible.

---

# 4. Intent Categories

Intent pages typically fall into several categories.

How-to queries

Examples:

how to become qa tester
how to become data analyst
how to start digital marketing career

Salary queries

Examples:

qa tester salary
data analyst salary
digital marketing salary

Career evaluation queries

Examples:

is qa a good career
is digital marketing worth it
is data analyst hard

Training queries

Examples:

how to learn qa testing
how to learn data analysis
how to start digital marketing

Each category targets a different stage of user decision making.

---

# 5. Page Content Structure

Intent pages should follow a consistent structure.

Introduction

Briefly address the user's question.

Explanation

Provide a clear answer to the query.

Career overview

Explain the profession briefly.

Next steps

Explain how users can start learning or training.

Call to action

Guide the user to the career page or quiz.

---

# 6. Avoiding Content Duplication

Intent pages must avoid duplicating the entire career page.

The career page is the main source of comprehensive information.

Intent pages should:

Answer the specific query
Provide focused information
Link to the career page for deeper details

Example:

Intent page

"QA Tester Salary"

Content focus:

salary ranges
factors affecting salary
career growth

Link to:

/career/qa-tester

---

# 7. Internal Linking

Intent pages should link to:

Career pages
Relevant guides
Relevant courses

Example linking:

Intent Page → Career Page
Intent Page → Quiz
Intent Page → Course comparison

Internal links should guide users deeper into the product funnel.

---

# 8. Intent Page Scale Strategy

Each career path can support multiple intent pages.

Example for QA Tester:

how-to page
salary page
career evaluation page
training page

Example set:

/career/how-to-become-qa
/career/qa-salary
/career/is-qa-good-career
/career/how-to-learn-qa

This structure allows the site to scale SEO traffic without creating duplicate content.

---

# 9. Relationship with the Quiz

Intent pages may include a call to action to take the quiz.

Example message:

"Not sure if this career fits you?
Take the quick career quiz."

This helps users who are still exploring options.

However, intent pages should not force the quiz as the only next step.

Users may also go directly to the career page.

---

# 10. Content Quality Requirements

Intent pages must:

Answer the search query clearly
Provide useful information
Avoid filler content
Link to deeper resources

Pages should remain concise and focused.

---

# 11. SEO Optimization

Intent pages should include:

Keyword-focused titles
Clear headings
Internal links
FAQ sections where relevant

Avoid:

Keyword stuffing
Thin content
Duplicate text from career pages

---

# 13. Scaling Content Production

Intent pages should be produced gradually.

Priority should be given to:

High search demand queries
Queries closely related to career decisions
Queries that lead to training decisions

Each new career path can generate several intent pages.

---

# 14. Integration with SEO Funnel

Intent pages play a key role in the SEO funnel.

Search Query

↓

Intent Page

↓

Career Page

↓

Courses

This helps move users from question-based searches toward training decisions.

---

# 15. Future Expansion

As the platform grows, more intent pages can be created.

Possible expansions include:

Advanced career questions
Industry-specific variations
Skill-based searches

However, all intent pages must remain connected to a core career page.
