Internal Linking Strategy
Project: משתחררים

Internal linking is critical for:

• SEO authority distribution
• helping search engines understand site structure
• guiding users through the decision process

The site follows a topic cluster model.

Main clusters:

Guides
Intent Pages
Career Paths
Courses
Checklist

---

CORE LINKING PRINCIPLE

Every page must guide the user one step deeper.

Typical flow:

Guide
→ Intent Page
→ Quiz
→ Career Page
→ Course Page

Alternative flow:

Intent Page
→ Career Page
→ Course Page

Checklist
→ Quiz
→ Career Page

---

GUIDE LINKING RULES

Guide pages are entry points.

They must link to:

Checklist
Quiz
Relevant careers
Relevant guides

Example

Guide:

מה לעשות אחרי הצבא

Links:

/checklist
/quiz
/career/qa-tester
/career/data-analyst

---

INTENT PAGE LINKING RULES

Intent pages must link to:

Career page
Quiz
Relevant guide

Example:

Intent:

איך להיות QA Tester

Links:

/career/qa-tester
/quiz
/guides/how-to-enter-tech

---

CAREER PAGE LINKING RULES

Career pages link to:

Courses
Relevant guides
Alternative careers

Example:

/career/qa-tester

Links:

/courses/qa-bootcamp
/courses/editorial/best-qa-courses
/guides/how-to-enter-tech

---

COURSE PAGE LINKING RULES

Course pages link back to:

Career page
Editorial course pages
Guides

Example:

/courses/qa-bootcamp

Links:

/career/qa-tester
/courses/editorial/best-qa-courses
/guides/how-to-enter-tech

---

EDITORIAL COURSE PAGE LINKING

These pages link to:

Individual course pages
Career page
Quiz

Example:

/courses/editorial/best-qa-courses

Links:

/courses/qa-bootcamp
/career/qa-tester
/quiz

---

HUB PAGES

Hub pages include:

/guides
/career
/courses
/checklist

These help search engines crawl the site.

---

CRAWL DEPTH RULE

Important pages must be reachable within 3 clicks from homepage.

Example:

Homepage
→ Guides
→ Article

Homepage
→ Career
→ QA Tester
