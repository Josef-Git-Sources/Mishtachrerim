\# strict-rtl-hebrew-ui-inspector



type: review-only



purpose:

inspect RTL correctness and Hebrew UI quality from code, markup, classnames, and UI structure.



use when:

\- UI files changed

\- Hebrew pages or components were updated

\- you want to catch RTL regressions, LTR leakage, or BiDi issues



primary focus:

\- global RTL setup

\- logical spacing vs hardcoded direction

\- directional icons and navigation

\- BiDi handling

\- Hebrew UI naturalness



does not do:

\- generic design critique

\- accessibility audit

\- auto-fixing in version 1



main file:

\- `SKILL.md`
